# 🚨 SafeLink – AI Powered Emergency Safety Platform

SafeLink is a futuristic emergency response and community safety platform built using **React, Vite, Tailwind CSS, Firebase, and Capacitor**.

🌐 Live Website:

The platform provides realtime emergency assistance, SOS alerts, NGO support, live safety monitoring, nearby help services, and community protection tools.

# 📲 Android App Setup

## Install Capacitor

```bash
npm install @capacitor/core @capacitor/cli
```

---

## Initialize Capacitor

```bash
npx cap init
```

---

## Add Android Platform

```bash
npm install @capacitor/android
```

```bash
npx cap add android
```

---

## Build React App

```bash
npm run build
```

---

## Sync Capacitor

```bash
npx cap sync
```

---

## Open Android Studio

```bash
npx cap open android
```
